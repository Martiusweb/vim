Function mappings					*mts-function-mappings*

<F1> Help
<F2> Display this cheat sheet
<F3> Toggle paste mode
<F5> Toggle Gundo
<F6> Display tagslist

------------------------------------------------------------------------------
Edition							*mts-edition*

- switch with previous line
Ã¨ switch with next line
H go to the first non-blank character of the line
L got to the end of the line

<C-D> Deletes current line in insert mode
<C-U> Uppercase current word

<C-space> Autocomplete language keywords

<leader>/  Disable search highlight
<leader>pi Toggle list invisible characters

<leader>en Toggle spell check and set English as current language
<leader>fr Toggle spell check and set French as current language
